# Style and Conventions

## Swift
- Swift 6.2+ required
- Platform: macOS 15+
- Uses structured concurrency (`try await`)

## TypeScript
- ES modules (`"type": "module"`)
- Strict TypeScript configuration
- ESLint for linting with stylistic plugin

## Code Style
- Follow existing patterns in the codebase
- Use descriptive variable names
- Type safety is enforced (no `any` or `unknown`)

## Project Conventions
- Lambda function name: `swift-lambda-example`
- Log group retention: 1 day
- Architecture: ARM64
- Runtime: provided.al2023
